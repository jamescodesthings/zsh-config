# PICO-8

## Keyboard Shortcuts

Toggle editor / console:
`ESC`

Run / reload cart:
`Ctrl+R`

Quick save:
`Ctrl+S`

Mute / unmute:
`Ctrl+M`

Previous / next editor tab:
`Alt + <- / ->`

Next tab:
`Ctrl+Tab`

Screenshot / label / GIF start / GIF save:
`Ctrl+6/7/8/9`

## Editor Shortcuts

Undo / redo:
`Ctrl+Z / Y`

Cut / copy / paste:
`Ctrl+X / C / V`

Search / find next:
`Ctrl+F / G`

Comment / uncomment block:
`Ctrl+B`

Duplicate line:
`Ctrl+D`

Help on keyword:
`Ctrl+U`

Jump to line:
`Ctrl+L`

Line start / end:
`Ctrl+W / E`

File start / end:
`Ctrl+up / dn`

Jump by word:
`Ctrl+<- / ->`

Indent / unindent:
`Tab / Shift+Tab`

Jump to prev / next function:
`Alt+up / dn`

## Runtime Shortcuts

Move (P1) / O+X buttons:
`D-pad / ZXCV`

O button (P1):
`Z, C, N`

X button (P1):
`X, V, M`

Pause menu:
`Enter / P`

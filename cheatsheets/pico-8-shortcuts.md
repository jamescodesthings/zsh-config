# pico-8-shortcuts

PICO-8 keyboard shortcuts for the editor and runtime.

## Global

- Toggle between editor and console:

`ESC`

- Run / reload cart:

`Ctrl+R`

- Quick save:

`Ctrl+S`

- Mute / unmute:

`Ctrl+M`

- Screenshot / label / GIF start / GIF save:

`Ctrl+6 / 7 / 8 / 9`

## Code Editor

- Undo / redo:

`Ctrl+Z / Y`

- Cut / copy / paste:

`Ctrl+X / C / V`

- Search / find next:

`Ctrl+F / G`

- Comment / uncomment block:

`Ctrl+B`

- Duplicate line:

`Ctrl+D`

- Help on keyword under cursor:

`Ctrl+U`

- Jump to line number:

`Ctrl+L`

- Line start / end:

`Ctrl+W / E`

- File start / end:

`Ctrl+up / dn`

- Jump by word:

`Ctrl+← / →`

- Indent / unindent:

`Tab / Shift+Tab`

- Jump to prev / next function:

`Alt+up / dn`

## Runtime

- Move (P1) / O+X buttons:

`D-pad / ZXCV`

- Pause menu:

`Enter / P`

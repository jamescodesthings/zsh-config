# zerocalc

> ZeroCalc keyboard shortcuts (PicoCalc device).

# Mouse Mode

- Toggle mouse mode:

`F5`

- Move cursor:

`D-pad`

- Left click:

`[`

- Right click:

`]`

# Navigation

- Home / End:

`RSHIFT + ← / →`

- Page Up / Down:

`RSHIFT + up / dn`

- File start / end:

`RSHIFT + [ / ]`

- Jump by word:

`Ctrl + ← / →`

- Word jump (readline):

`Alt + F / B`

# Selection

- Select character:

`LSHIFT + arrow`

- Select word:

`LSHIFT + RSHIFT + ← / →`

- Select to line start / end:

`LSHIFT + RSHIFT + [ / ]`

# Right Shift Shortcuts

- Line start:

`RSHIFT + A`

- Line end:

`RSHIFT + E`

- Delete to line start:

`RSHIFT + U`

- Delete to line end:

`RSHIFT + K`

- Delete word back:

`RSHIFT + W`

- Clear screen:

`RSHIFT + L`

- Interrupt:

`RSHIFT + C`

- EOF:

`RSHIFT + D`

- Suspend:

`RSHIFT + Z`

- Send Ctrl+key:

`RSHIFT + key`

- Send Ctrl+Shift+key:

`LSHIFT + RSHIFT + key`

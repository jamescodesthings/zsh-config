# zerocalc

ZeroCalc keyboard shortcuts (PicoCalc device).

## Mouse Mode

- Toggle mouse mode:

`F5`

- Move cursor:

`D-pad`

- Left click:

`[`

- Right click:

`]`

## Navigation

- Home / End:

`RSHIFT + ← / →`

- Page Up / Down:

`RSHIFT + up / dn`

- File start / end:

`RSHIFT + [ / ]`

- Jump by word:

`Ctrl + ← / →`

- Word jump (readline):

`Alt + F / B`

## Selection

- Select character:

`LSHIFT + arrow`

- Select word:

`LSHIFT + RSHIFT + ← / →`

- Select to line start / end:

`LSHIFT + RSHIFT + [ / ]`

## Right Shift Shortcuts

| Keys | Action |
| --- | --- |
| `RSHIFT + A` | Line start |
| `RSHIFT + E` | Line end |
| `RSHIFT + U` | Delete to line start |
| `RSHIFT + K` | Delete to line end |
| `RSHIFT + W` | Delete word back |
| `RSHIFT + L` | Clear screen |
| `RSHIFT + C` | Interrupt |
| `RSHIFT + D` | EOF |
| `RSHIFT + Z` | Suspend |
| `RSHIFT + key` | Ctrl+key |
| `LSHIFT + RSHIFT + key` | Ctrl+Shift+key |

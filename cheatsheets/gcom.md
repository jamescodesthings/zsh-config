# gcom

> Check out the gitflow main branch.

## Usage

- Check out the main branch configured for this repo:

`gcom`

## Notes

- Reads branch name from `git config gitflow.branch.main`
- Defaults to `main` if not configured
- Configure with: `git config gitflow.branch.main main`

## Related commands

- Check out develop branch:
`cheat gcod`

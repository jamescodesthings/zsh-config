# gcod

> Check out the gitflow develop branch.

## Usage

- Check out the develop branch configured for this repo:

`gcod`

## Notes

- Reads the branch name from `git config gitflow.branch.develop`
- Configure it with: `git config gitflow.branch.develop develop`
- Errors if no develop branch is configured

## Related commands

- Check out main branch:
`cheat gcom`
